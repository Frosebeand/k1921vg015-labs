# Лабораторные работы по микроконтроллеру К1921ВГ015

## О проекте

Данный репозиторий содержит выполненные лабораторные работы по курсу программирования микроконтроллеров на базе отечественного RISC-V МК К1921ВГ015 (АО «НИИЭТ»).

## Содержание

| Лабораторная работа | Описание |
|---------------------|----------|
| Лаба 1 | Первое включение, мигание светодиодом (программная и аппаратная задержки) |
| Лаба 2 | Кнопка WAKEUP0, бегущий огонь с переключением скорости |
| Лаба 3 | ШИМ на двух каналах таймера TMR1, изменение скважности |
| Лаба 4 | RTC как часы, вывод времени по UART |
| Лаба 5 | Режимы энергопотребления PMU (IDLE, STOP, POWEROFF) |
| Лаба 6 | АЦП и внутренний термодатчик, вывод температуры |
| Лаба 7 | Генератор случайных чисел TRNG, чтение FIFO |
| Лаба 8 | UART-консоль, обработка команд (retarget) |
| Лаба 9 | RTC-будильник, пробуждение из POWEROFF, циклический перебор светодиодов |
| Лаба 10 | Игра «Реакция» (TRNG + таймер + UART) |
| Лаба 11 | Работа с Flash-памятью (стирание, запись, чтение) |
| Лаба 12 | UART + DMA (автоматическая передача блока данных) |

## Инструменты

- VS Code + расширение **NIIET Aspect**
- OpenOCD (JTAG-отладка)
- Zadig (установка драйвера WinUSB для JTAG)
- RISC-V toolchain (xpack-riscv-none-elf-gcc)

## Аппаратная платформа

- Отладочная плата КФДЛ.441461.029 (или совместимая)
- Микроконтроллер К1921ВГ015 (RISC-V, 50 МГц, 1 МБ Flash, 256+64 КБ ОЗУ)

## Запуск

1. Клонировать репозиторий
2. Открыть папку с проектом в VS Code
3. Настроить расширение NIIET Aspect (указать пути к SDK и toolchain)
4. Собрать проект (Build)
5. Запустить отладку (F5)
